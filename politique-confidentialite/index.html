6.3 Vente de données

7. VOS DROITS RGPD

Vous disposez des droits suivants sur vos données personnelles :

7.1 Droit d'accès (Article 15 RGPD)

Vous pouvez demander une copie de toutes vos données personnelles. Nous fournirons les données dans un format structuré et lisible.

7.2 Droit de rectification (Article 16 RGPD)

Si vos données sont inexactes ou incomplètes, vous pouvez les corriger. Envoyez-nous une demande avec les corrections.

7.3 Droit à l'oubli (Article 17 RGPD)

Vous pouvez demander l'effacement de vos données personnelles (exceptions : obligations légales, intérêt public). Nous supprimerons vos données dans les 30 jours.

7.4 Droit à la portabilité (Article 20 RGPD)

Vous pouvez recevoir vos données dans un format standard (CSV, JSON) et les transférer à un autre service.

7.5 Droit d'opposition (Article 21 RGPD)

Vous pouvez vous opposer à l'utilisation de vos données pour certaines finalités (marketing, analytics). Une case à cocher est disponible lors de l'inscription.

7.6 Droit à la limitation (Article 18 RGPD)

Vous pouvez demander l'arrêt temporaire du traitement de vos données (sauf obligations légales).

7.7 Comment exercer vos droits

Envoyez une demande écrite à :

Email : dpo@finances-conseil-france.fr
Objet : [RGPD] Demande d'exercice de droit

Veuillez préciser:
• Votre droit (accès, rectification, oubli, portabilité, opposition)
• Votre email ou numéro client
• Si possible: copies de justificatif d'identité

Délai de réponse : 30 jours (extensible à 60 jours pour les demandes complexes)

Un cookie est un petit fichier texte stocké sur votre appareil. Il permet à un site web de vous reconnaître lors de visites ultérieures.
